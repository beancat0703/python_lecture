Q2: Digit Distance

对于给定的整数，digit distance（数字距离） 是连续数字之间绝对差的总和。例如：

  • 61 的 digit distance 是 5，因为 |6 - 1| = 5。
  • 71253 的 digit distance 是 12（|7-1| + |1-2| + |2-5| + |5-3| = 6 + 1 + 3 + 2）。
  • 6 的 digit distance 是 0，因为没有连续的两位数。

编写一个函数计算正整数的 digit distance。必须使用递归，否则测试将失败。

Q3: Interleaved Sum

例如，interleaved_sum(5, lambda x: x, lambda x: x * x) 返回 1 + 2*2 + 3 + 4*4 + 5 = 29。

提示： 引入一个内部辅助函数，接收一个奇数 k，计算从 k 到 num 的交叉和；或者使用互递归（mutual recursion）。

Q6: Deep Map

定义：嵌套数字列表 是一个包含数字和列表的列表。它可以只包含数字、只包含列表或两者的混合。列表本身也必须是嵌套数字列表。例如：[1, [2, [3]], 4]、[1, 2, 3] 和 [[1, 2], [3, 4]] 都是嵌套数字列表。

编写函数 deep_map，接收两个参数：嵌套数字列表 s 和单参数函数 f。它就地修改 s，将 s 中的每个数字替换为对该数字调用 f 的结果。

⚠ 重要
deep_map 返回 None，不应创建任何新列表。
💡 提示
type(a) == list 在 a 是列表时求值为 True。
def deep_map(f, s: list) -> list: """Replace all non-list elements x with f(x) in the nested list s. >>> six = [1, 2, [3, [4], 5], 6] >>> deep_map(lambda x: x * x, six) >>> six [1, 4, [9, [16], 25], 36] >>> # Check that you're not making new lists >>> s = [3, [1, [4, [1]]]] >>> s1 = s[1] >>> s2 = s1[1] >>> s3 = s2[1] >>> deep_map(lambda x: x + 1, s) >>> s [4, [2, [5, [2]]]] >>> s1 is s[1] True >>> s2 is s1[1] True >>> s3 is s2[1] True """ "*** YOUR CODE HERE ***"
