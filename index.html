Lab 0 Setup

一、检查 Python 是否安装成功以及版本

键盘按 ⌘ + Space。

输入 terminal，打开终端。

输入 python3 --version，检查 Python 版本（新版本 macOS 自动安装 Python 3）。

打开 VS Code，尝试新建 .py 文件并运行

打开 VS Code。

点击打开文件夹，在桌面新建一个文件夹，命名为 python。创建完成后打开。

在屏幕左上角找到这个按钮，并点击。

新建文件命名为 helloworld.py（注意不要漏了后缀名 .py）。

点击这个文件，在右侧编辑区域输入代码。

# helloworld.py print("Hello, world!")
⚠ 重要： 修改完代码后，务必按 Cmd + S（Mac）或 Ctrl + S（Windows）保存文件，否则终端运行的是修改前的旧代码。

在屏幕顶部找到任务栏，点击 终端 → 新建终端。

可以在下侧看到这样的一栏，称为终端（Terminal）。

在终端输入

💡 提示： 代码前面的 $ 符号表示这是终端命令提示符，不需要你手动输入。你只需要输入 $ 后面的内容即可。
$ python3 helloworld.py

并回车运行。

可以看到终端中出现这样的样式，则环境配置成功。

环境配置成功！你已经完成了 Python 开发环境的搭建。

💡 实用小贴士 (Tips)

💻 1. Windows 系统运行 Python

如果你是 Windows 系统，在终端中输入 python3 可能无法运行。请尝试使用 python 或 py 替代。例如：
C:\Users\... python ok -q a_plus_abs_b
🔌 2. Ok 本地测试（无需联网）

第一次使用 Ok 测试时，终端可能会提示你输入邮箱。如果输入邮箱后仍无法进行测试（例如网络问题），可以在 Ok 命令后添加 --local 参数来本地运行测试：
$ python3 ok -q a_plus_abs_b --local
这样 Ok 就不会尝试连接服务器，直接在本机运行测试。
🧪 3. 交互式测试代码

在 VS Code 写作业时，你可以在终端中使用 -i 参数进入 Python 交互式环境，方便随时测试你写的函数：
$ python3 -i hw01.py
进入后，你会看到 >>> 提示符，可以直接调用你定义的函数来测试结果。例如：
>>> a_plus_abs_b(2, -3) 5 >>> hailstone(10) 10 5 16 8 4 2 1
修改代码后重新加载：输入 exit() 退出，然后再次运行 python3 -i hw01.py 即可。这个技巧非常实用，推荐你养成习惯！
